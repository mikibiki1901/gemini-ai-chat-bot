# 🤖 Moj AI Python Mentor

Ovo je moja prva Python aplikacija! Koristim je da učim programiranje uz pomoć Google Gemini AI modela.

## ✨ Karakteristike
- **AI Model:** Gemini 2.5 Flash
- **Funkcija:** Interaktivni mentor koji postavlja zadatke i objašnjava Python koncepte.
- **Tehnologije:** Python, `google-genai` biblioteka, `.env` za bezbednost ključeva.

## 🚀 Kako pokrenuti projekat
1. Kloniraj repozitorijum.
2. Instaliraj potrebne biblioteke: `pip install google-genai python-dotenv`.
3. Napravi `.env` fajl i unesi svoj `GEMINI_API_KEY`.
4. Pokreni aplikaciju komandom:
   ```bash
   python moj_ai.py
